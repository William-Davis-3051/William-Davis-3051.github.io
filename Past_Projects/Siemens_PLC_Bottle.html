Brief Project Description: This project was designed to mimic a conveyor bottle filler and rejection system. An HMI was used to start, stop, and pause the conveyor system as well as have a reject button to reject bottles with defects.

Hard Part of the Project: This was the 1st real interactive I had with PLCs so being able to understand and put together the Ladder Logic for the whole system was difficult do to all the moving parts. However, the hardest part was not filling a rejected bottle so telling the system that the bottle no longer exist was really fun to try and get right as well as keeping track of the number of filled and rejected bottles
